# Contribution Guidelines

## How to contribute?

**Repository:** [https://github.com/nhnquang11/dots](https://github.com/nhnquang11/dots).

Repository is open to all members of the community. Follow the steps below to contribute to this project.

1. Fork the repository
2. Clone the repository to your local machine
3. Create a branch
4. Make necessary changes and commit those changes
5. Push changes to GitHub
6. Submit your changes for review
